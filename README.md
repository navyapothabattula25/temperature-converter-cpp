# Temperature Converter (C++)

This is a C++ project that converts temperature between:
- Celsius
- Fahrenheit
- Kelvin

## Features
- User input validation
- Conversion to all temperature units
- Clean formatted output using iomanip

##  Technologies Used
- C++
- iostream, iomanip, cctype

##  How to Run
1. Compile:
   g++ temperature.cpp -o temp
2. Run:
   ./temp
